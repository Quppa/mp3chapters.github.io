Online MP3 Podcast Chapter Editor GitHub

Add chapters to your podcast MP3 files.

A simple online tool without install or signup, running locally in your browser. Works with all podcast players that support id3 chapters.

To start, load your audio file.

Edit chapters

Chapters are managed using the same syntax also used by YouTube. Each chapter is one line, starting with the timestamp when the chapter begins, followed by a space, followed by the name of the chapter. The first chapter should start at 00:00. The last chapter will end when the MP3 file ends.

You can add and edit chapters manually below. You can also go to a desired starting point in the player and click the Add Chapter button to start a chapter at that timestamp.

Tip: Copy and paste the list of chapters into your episode description, which some players like Spotify use.

Edit MP3 tags

MP3 files can contain metadata such as title, artist, etc., known as ID3 tags. These tags are displayed by most podcast players. You can edit these tags below. For podcasts, not all fields are applicable and can be left empty.

Podlove Simple Chapters XML

An alternative to embedding chapters directly in the audio file is the Podlove Simple Chapters format using XML, which can be embedded in the <item> tag of a podcast RSS feed. While most podcast players use embedded chapters, some other players (notably Spotify) only support time stamps in the episode description or podlove chapters.

If you wish to add time stamps to your episode description, just copy and paste your chapter list. If you wish to use Podlove chapters, use the code below. See the Podlove documentation for more details.
